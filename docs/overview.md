# QRNG Overview

## What is QRNG?
A Quantum Random Number Generator (QRNG) generates random numbers using quantum phenomena, providing true randomness unlike traditional PRNGs.

## Key Differences from PRNGs
- **Source**: Quantum phenomena vs mathematical algorithms
- **Speed**: Slower but more secure
- **Predictability**: Truly unpredictable
- **Use Cases**: Cryptography, security, simulations
